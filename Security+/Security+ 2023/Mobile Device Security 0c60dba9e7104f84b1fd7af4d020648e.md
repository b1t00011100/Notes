# Mobile Device Security

### Securing Wireless Devices

- Two main Types of Wireless Connectivity
    - **Wi-fi**
    - **Bluetooth**
- If wifi is in use, it should be to set to **WPA2(WiFi Protected Access 2)** is the highest level of wireless security.
    - It relies on AES
- Bluetooth pairing creates a shared link key to encrypt the connection.
- Wired Devices are almost always more secure than wireless ones.

### Mobile Malware

- Securing Phone from Malware
    - Anti-Virus Software
    - Getting Patches and Updates
    - Only install apps from App Store or Play Store.
- **Mobile Malware**
    - Do not Jailbreak/root your device
    - Don’t use custom firmware or a custom ROM
    - Only load official store apps
    - Always update your phone’s operating system
    

### SIM Cloning & ID theft

- **SIM** : Subscriber Identity Module
    - Integrated circuit that securely stores the international mobile subscriber identity (IMSI) number and its related key.
- **SIM Cloning** : Allows two phones to utilize the same service and allows an attacker to gain access to the phones data.
- SIM v1 cards were easy to clone but SIM v2 is much harder.

### Bluetooth Attacks

- **Bluejacking** : Sending of unsolicited messages to Bluetooth enabled devices.
    - Sending information to a device
- **Bluesnarfing** : Unauthorized access of information from a wireless device over a Bluetooth connection.
    - Taking information from a device.

### Mobile Device Theft

- Always ensure device is backed up.
- Full disk Encryption
- Tracking GPS
- **Remote Lock** : Requires a pin or password before someone can use the device
- **Remote Wipe** : Remotely erases the contents of the device to ensure the information is not recovered by the thief

### Security Of Apps

- Only install Apps from the official Stores
- Jailbreaking/Rooting : Removing default Security Protections on the phone.
- While Browsing, Ensuring secure version of Http is used
- **Mobile Device Management** : Centralized Software solutions that allows system administrator to create and enforce policies across its mobile devices.
- Turn location services off to ensure privacy.
- **Geotagging** : Embedding of the geolocation coordinates into a piece of data .(i.e a photo)
- Geotagging should be considered when developing your organizations security policies.

### BYOD

- **B**ring **Y**our **O**wn **D**evice
- BYOD introduces a lot of security issues to consider.
- **Storage Segmentation** : Creating a clear separation between personal data and company data on your device.
- **CYOD** (Choose Your Own Device) model

### Hardening Mobile Device

- 10 ways to increase Mobile Security
    1. Update your device to use to latest version software.
    2. Install Anti-Virus
    3. Train users on proper security and use of the device
    4. Only install application from the official Stores 
    5. Do not root or jailbreak device
    6. Only use SIM v2 on your devices
    7. Turn off all unnecessary features
    8. Turn on encryption on voice and text data
    9. Use Strong password or Biometrics
    10. Don’t Allow BYOD 
    
- Ensure your organization has a good security policy for mobile devices.
